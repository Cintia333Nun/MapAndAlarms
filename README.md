<h1>***Alarms and Map Generic Project*** <br/></h1>
<br/>Need add your "json google services" and "api key"<br/>
 <br/>Technologies Used:<br/>
● Gradle with  Kotlin and news Cats
● Java  Kotlin
● Jetpack Compose 
● Views
● Clean Architecture (MVVM)   
● Flows
● Room
● Sqlite
● ViewBinding
● LiveData
● Maps api
● Geocoding
● Alarm Manager
● Dagger/Hilt
● SharePreference
● Notification
● BrodcastReceiver
● Permissions
● Picker Time And Date
